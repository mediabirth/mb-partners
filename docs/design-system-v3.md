# デザインシステム v3「墨と絹 × インディゴ」

> これは色相の変更ではなく **適用思想の変更** である。Media Birth インディゴ（`#4733E6`）はブランド資産として色を変えず保持し、その使われ方を「AIのデフォルト」から「ブランドの署名」へ変える。
> 相互参照: [copy-guideline](./copy-guideline.md)（言葉の規律）と対になる、色と形の規律。

## 教義（8箇条）

1. **基調** — 紙白（暖色ニュートラルランプ）× 墨黒。無彩色ニュートラルは全廃し暖色ランプへ。基準紙色 `--paper #F6F4EE`、墨 `--ink #1B1A17`。
2. **角の思想** — 角丸ほぼゼロ。カード/入力/ボタン = 4px（`--r-card`/`--r-el`）。**999px ピルは体系として廃止**（`--radius-pill` も 4px）。
3. **インディゴの使用規律** — 使用量を絞るほど色は強くなる。
   - **許可**: 金額の下線数字（RewardFigure）・テキストリンク・選択状態・フォーカスリング・ステータスの現在ドット・ロゴ/ワードマーク。
   - **禁止**: 薄紫ピル（bg-accent 面）・塗りボタン・アイコン地・装飾グラデーション。
4. **CTA = 墨黒の塗りボタン**（`fill-primary = --ink`）。1画面1つの primary 規律は継続。secondary/ghost は墨テキスト。
5. **数字** — 全金額・件数は等幅（`tnum`）。
6. **署名コンポーネント** — `RewardPill` を廃止し **`RewardFigure`**（インディゴの下線付き等幅数字）へ。情報ピル → 小さな角 tag（2px 角・0.5px アウトライン・地は淡く）。ステータスは dot+text（選択ドットはインディゴ）。
7. **書体** — 本バッチでは触れない。`--font-display` は明朝 display への差し替え口として温存（現状は sans=Zen にエイリアス）。
8. **静音化 5 原則・v2.2 の構造規律**（0.5px 罫線・weight 400/500・階層・説明文は操作の瞬間のみ）は骨格として全て存続。

| 紙白カード | `--bg` / `--s-0` | `#FAF7F0` | `#201E22` |

| tag地（旧薄紫） | `--blue-bg` / `--blue-bg2` | `#ECE6DA` / `#F1EDE2`（暖色退役） | `#26242B` / `#201E24` |

| 影 | `--shadow*` | 墨系の淡い影に軽量化 | 同 |

- **トークン化率 ~94%** のため `:root` 差替で 3 面（app/console/vendor）が自動追随する。
- **ダークモード**は `:root[data-theme="dark"]` の opt-in（`prefers-color-scheme` 自動適用はしない）＝ライブ無影響。トークン化された面/文字/線は追随するが、ハードコード `#fff` カードは別バッチで token 化してから全面点灯する（本バッチは基盤定義＋実機スクショ検証まで）。

## 署名コンポーネント

- **`components/ui/RewardFigure.tsx`** — 報酬 = インディゴ下線付き等幅数字（地色なし・角丸なし）。`RewardPill` は後方互換の再エクスポート（実体は RewardFigure）。
- **`.chip`（globals.css）** — 2px 角・0.5px アウトライン・地は透明。色相は系統維持（紹介=藍/協力=ティール/直販=中立）。
- **`.ui-tag` / StatusPill / SettingsRow** — `--radius-pill`（4px）を参照＝ピル廃止に自動追随。選択時は淡い ink 地 + インディゴ境界/文字（許可）。
- **`.ui-btn--primary` / `.btn-p`** — `--ink` 塗り。secondary/ghost は墨テキスト。

## 適用範囲（本バッチ）

APP・コンソール・ベンダー・公開ページ（`/r/[token]`・`/book`・`/legal`・login/invite）・メールテンプレ（`lib/mail-render.ts`・`lib/email.ts`：CTA塗り→墨・パネル→紙白・角4px、ワードマーク/リンクのインディゴは署名として維持）・PWA（manifest `theme_color`/`background_color` → 紙白 `#F6F4EE`、splash 紙白化）。ロゴはインディゴ保持（ブランド継続＝色替え不要）。

## 不変（衣装のみ・機能非変更）

機能・動線・情報設計は一切変更しない。money（計算意味・確定ガード・snapshot）非接触、session 独立、identity、3 面分離は全て不変。
