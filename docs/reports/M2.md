| ルート | メソッド | 説明 |

## 数値検証

源泉所得税計算:

```
gross       = ¥80,000
withholding = Math.round(80000 × 0.1021) = Math.round(8168) = ¥8,168
net         = 80000 − 8168 = ¥71,832
```

`reward_snapshot` はフォーム送信時にサービスメニューから取得されDB に保存される（後からメニューが変更されても過去の報酬額は保持）。

## 主な修正点

- API ルートを `/api/` (プロジェクトルート) から `app/api/` (Next.js App Router が認識するディレクトリ) に移動
- `createServiceRoleClient()` の `await` 漏れを修正（referral API）
- `profiles.role` の正しい値: `'owner'` / `'partner'`（`'admin'` は無効）
- コンソール API の認可チェック: `role === 'partner' || !profile` で判定
