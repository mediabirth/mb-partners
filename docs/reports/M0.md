## 技術メモ

- `lib/supabase/server.ts` は `@supabase/ssr` の `createServerClient` を使用し、Cookie 読み書きを Next.js の `cookies()` API に委譲
- `lib/supabase/client.ts` は `createBrowserClient` でシングルトン管理
- `tsconfig.json` に `"@/*": ["./*"]` のパスエイリアスを設定
