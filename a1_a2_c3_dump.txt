■ C-3 — パートナー詳細（/console/partners/[id]）
========================================================

[BEFORE]
  - 基本情報カード (avatar/name/code/status/tax/KYC)
  - 累計案件 (count only)
  - 口座変更申請パネル

[AFTER — v9 style]

  ヘッダーカード:
    大きめavatar + 名前 + コード + ステータスpill + KYC + メール + 登録日 + 税区分

  KPI ストリップ (4カード):
    累計報酬(税込) | 累計案件 | 進行中 | 今月成約

  2カラムレイアウト:
    左 (広):
      ■ 案件一覧 (N件)
        ステータスchip + 顧客名 + サービス名 + 日付 + 報酬額
        (最大30件表示)

      ■ 振込履歴 (payout_items)
        月 | ステータス | 総額 | 源泉 | 振込額
        (最大24件)

    右 (320px):
      ■ 基本情報 (税区分, KYC, コード)
      ■ 問い合わせ → 件数リンク (/console/inquiries?partner_id=X)
      ■ 振込口座 (bank_name + branch + type + number + holder)
      ■ 口座変更申請 (BankChangePanel — 既存)

[追加クエリ — 2ラウンド構成]
  Round 1 (並列): auth check + partner(+profile_id) + bank_change_requests
  Round 2 (並列): deals + payout_items + inquiries count

  deals: SELECT id, customer_name, status, amount, created_at, services(name)
  payout_items: SELECT id, gross, withholding, net, payout_batches(month, status)
  inquiries: COUNT WHERE partner_id = id
